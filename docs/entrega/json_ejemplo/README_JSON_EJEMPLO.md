# JSON de ejemplo

Esta carpeta contiene archivos JSON preparados para entregar junto al proyecto. Los archivos originales de ejecucion se mantienen en sus rutas habituales.

## Archivos

- `json_prueba.json`: ejemplo documental pequeño y legible. No pretende sustituir al guardado real completo; sirve para mostrar la forma general de jugador, enemigos, objetos, puertas y estado.
- `cuevas_ejemplo.json`: copia de `datos/cuevas.json`. Muestra la configuracion inicial real de la mazmorra.
- `partida_guardada_ejemplo.json`: ejemplo de estado persistido incluido en esta carpeta de entrega.
- `ranking_ejemplo.json`: copia de `ranking.json`. Muestra el formato del ranking local.

## Uso en la entrega

Para defender la parte de persistencia, el archivo mas importante es `cuevas_ejemplo.json`, porque demuestra que el mapa, enemigos, objetos y conexiones se cargan desde datos externos. `partida_guardada_ejemplo.json` y `ranking_ejemplo.json` sirven como ejemplos adicionales de guardado/carga y resultados finales.
